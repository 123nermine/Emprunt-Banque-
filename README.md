# Calculateur d'emprunt bancaire (Pro

Ce programme en C++ permet de comparer les mensualités
entre plusieurs banques :
- Crédit Agricole
- BNP Paribas
- LCL
- Crédit Mutuel

## Fonctionnalités
- Calcul des mensualités
- Comparaison de 4 banques
- 2 taux par banque ( 3% et 4%)
- 3 durées de prêt

##  Exécution
Compiler avec :
g++ main.cpp -o emprunt

Puis :
./emprunt

## Exemple
Affichage sous forme de tableau clair et le stocker dans un fichier csv avec les mensualités.
